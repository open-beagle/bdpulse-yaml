# Bdpulse Yaml

Bdpulse Yaml 是 [BDPulse (原 Drone)](https://github.com/open-beagle) CI/CD 引擎的YAML 解析器。

## 简介

专门用于解析和校验 BDPulse/Drone 流水线 YAML 配置文件（如 `.beagle.yml` 或 `.drone.yml`）的模块。

```bash
go get github.com/open-beagle/bdpulse-yaml
```
